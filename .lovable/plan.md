

# خطة: iRecycle Health Live — المحادثة الصوتية مع المدرب الصحي

## الفكرة
إضافة تبويب جديد "Live" في iRecycle Health يتيح للمستخدم التحدث صوتياً مع المدرب الصحي AI والاستماع لردوده منطوقة — محادثة صوتية حية ثنائية الاتجاه.

## كيف يعمل تقنياً

```text
┌──────────────┐    Web Speech API     ┌──────────────┐
│  المستخدم     │ ──── STT ──────────► │  نص السؤال   │
│  يتكلم 🎤    │                       │              │
└──────────────┘                       └──────┬───────┘
                                              │
                                              ▼
                                   ┌────────────────────┐
                                   │ ai-unified-gateway  │
                                   │ (Gemini 2.5 Flash) │
                                   └──────────┬─────────┘
                                              │
                                              ▼
                                       ┌──────────────┐
                                       │  نص الرد     │
                                       └──────┬───────┘
                                              │
                                    Web Speech API (TTS)
                                              │
                                              ▼
                                   ┌──────────────────┐
                                   │ المستخدم يسمع 🔊 │
                                   └──────────────────┘
```

**المقاربة**: استخدام **Web Speech API** المدمجة في المتصفح (بدون مكتبات خارجية):
- **STT**: `SpeechRecognition` لتحويل كلام المستخدم لنص
- **TTS**: `SpeechSynthesis` لنطق رد الذكاء الاصطناعي بالعربية
- **AI**: البوابة الموحدة الموجودة (`ai-unified-gateway`) مع Gemini
