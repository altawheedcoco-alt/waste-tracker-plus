

# خطة: إعادة تسمية "الصفحة التجارية" إلى "صفحة المنظمة" + تجميع التبويبات المتشابهة

## ١. إعادة التسمية في كل الأماكن

| الملف | التغيير |
|-------|---------|
| `OrganizationProfile.tsx` | تغيير label التبويب + تعليقات الكود + ErrorBoundary fallback |
| `sidebarConfig.ts` | تغيير `labelAr` من "الصفحة التجارية" إلى "صفحة المنظمة" + اسم المجموعة |
| `BusinessPageSharePanel.tsx` | تغيير عنوان "مشاركة الصفحة التجارية" → "مشاركة صفحة المنظمة" |
| `PartnersView.tsx` | تغيير عنوان الديالوج |

## ٢. إعادة تنظيم التبويبات في مجموعات واضحة

التبويبات الحالية (15 تبويب) متراصة بدون تجميع بصري واضح. سيتم تقسيمها إلى 4 مجموعات:

```text
┌─────────────────────────────────────────────────┐
│  📌 الهوية والعرض                                │
│  [صفحة المنظمة] [البورتفوليو] [الملف التجاري]   │
│  [المنشورات]                                     │
├─────────────────────────────────────────────────┤
│  📋 البيانات والتواصل                            │
│  [البيانات الأساسية] [الممثلون] [التواصل]        │
│  [الموقع والصور]                                 │
├─────────────────────────────────────────────────┤
│  📄 الوثائق والتحليل                             │
│  [الوثائق] [تحليل الوثائق] [تحليل الجهة]        │
├─────────────────────────────────────────────────┤
│  🔐 التوثيق والمصداقية                           │
│  [الختم والتوقيع] [الشهادات] [الإفادة] [المشاركة]│
└─────────────────────────────────────────────────┘
```

## ٣. تحسين V2TabsNav لدعم المجموعات

إضافة خاصية `group` اختيارية لـ `TabItem` بحيث تظهر فواصل بصرية (separator + label) بين المجموعات داخل شريط التبويبات.
