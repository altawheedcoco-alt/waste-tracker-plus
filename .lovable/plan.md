

# تحليل شامل: المتشابهات المتناثرة + خطة إعادة التنظيم

بعد فحص كل الـ 1467 سطر في `sidebarConfig.ts`، وجدت **7 مجالات رئيسية** فيها تكرار وتشتت:

## الحل المقترح: دمج المتشابهات

### الاقتراح A: "الدمج الجريء" — تقليل المجموعات بشكل كبير

```text
قبل: ~50 مجموعة
بعد: ~30 مجموعة

التغييرات الرئيسية:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. دمج كل الطباعة → مجموعة "مركز الطباعة والتصدير"
   يضم: مركز الطباعة + التقرير التجميعي + تصدير Excel
   + طباعة الشهادات + طباعة الفواتير + طباعة المانيفست

2. دمج كل التوقيعات → مجموعة "مركز التوقيعات"
   يضم: صندوق التوقيعات + التوقيعات الرقمية
   + QR/باركود + التحقق والأمان + ختم المستندات

3. دمج كل التقارير → مجموعتين فقط:
   • "التقارير التشغيلية" = شحنات + سجلات + تجميعي
   • "التقارير البيئية والمالية" = بيئة + كربون + ESG + مالية

4. دمج كل التتبع → مجموعة "مركز التتبع والخرائط"
   يضم: تتبع شحنات + تتبع سائقين + خرائط + مواقع محفوظة

5. دمج التواصل الاجتماعي → مجموعة "التواصل والمنشورات"
   يضم: دردشة + مجموعات + بث + منشورات + stories + أخبار
```

### الاقتراح B: "التنظيم المحافظ" — إبقاء المجموعات مع تحسين الأقسام

```text
قبل: 14 قسم بتوزيع غير مثالي
بعد: 14 قسم بتوزيع أدق

التغييرات:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• نقل "مركز الطباعة" من العمليات → قسم المستندات
• نقل "Stories" من الطلبات → قسم التواصل
• نقل "تقارير التخلص" → قسم التقارير
• نقل "السلامة" → مجموعة مستقلة في قسم العمليات
• لا حذف أو دمج — فقط إعادة ترتيب
```

## ما سيتم تنفيذه (الاقتراح A — الدمج الجريء)

### الملفات المتأثرة:
1. **`src/config/sidebarConfig.ts`** — إعادة هيكلة المجموعات: دمج المتشابهات، إعادة توزيع العناصر
2. **`src/config/sidebarConfig.ts`** — تحديث `SIDEBAR_SECTIONS` و `CATEGORY_ORDER`

### لا تغييرات في:
- قاعدة البيانات
- المسارات (paths) — كلها تبقى كما هي
- الصفحات نفسها — فقط طريقة الوصول إليها من القائمة

### التغييرات المحددة:

| المجموعة الجديدة | تدمج من | عدد العناصر |
|---|---|---|
| **مركز الطباعة والتصدير** | transporter-manual(طباعة) + docs-certs-invoices(طباعة) + shipment-reports(تجميعي) | ~6 |
| **مركز التوقيعات والتوثيق** | docs-signatures + signing-inbox items | ~5 |
| **التقارير الشاملة** | shipment-reports + waste-registers + waste-analysis → مجموعة واحدة | ~9 |
| **التقارير البيئية والاستدامة** | environmental-reports (تبقى كما هي) | 4 |
| **التتبع والخرائط** | generator-tracking + fleet-tracking + maps → مجموعة واحدة | ~6 |
| **التواصل والمنشورات** | direct-messages + stories item + آخر الأخبار | ~7 |
