

# خطة: مركز التواصل والمشاركة الموحد (Communication Hub)

## إعادة صياغة الطلب

تحويل مجموعة "التواصل والطلبات" (الحالية في الترتيب 15) إلى **أول مجموعة في القائمة الجانبية ولوحة التحكم**، مع إعادة هيكلتها لتصبح مركز تواصل شامل يضم كل قنوات الاتصال والمشاركة (العامة والخاصة) في مكان واحد.

## التغييرات المطلوبة

### 1. إعادة هيكلة مجموعة التواصل في `sidebarConfig.ts`

نقل مجموعة `communication` من الترتيب 15 إلى **الترتيب 1** (قبل "المنظمة والهيكل")، مع إعادة تسميتها وتوسيعها:

**الاسم الجديد:** "التواصل والمشاركة" / "Communication Hub"

**العناصر المُعاد تنظيمها:**
| العنصر | المصدر الحالي | ملاحظة |
|--------|-------------|--------|
| الرسائل (Chat) | communication | يبقى |
| الإشعارات | لا يوجد في sidebar حالياً | **يُضاف** - ينقل من MobileBottomNav فقط لصفحة sidebar |
| مركز الملاحظات | smart-tools (ترتيب 17) | **يُنقل هنا** |
| الحالات (Stories) | communication | يبقى |
| الاجتماعات المرئية | communication | يبقى |
| طلباتي | communication | يبقى |
| آخر أخبار الشركاء | org-structure | **نسخة مرجعية هنا** |
| بوابة العملاء | communication | يبقى |

### 2. ودجت "مركز التواصل" في لوحة التحكم الرئيسية

إنشاء `src/components/dashboard/widgets/CommunicationHubWidget.tsx`:
- بطاقة مدمجة تعرض: عدد الرسائل غير المقروءة، الإشعارات المعلقة، الملاحظات الأخيرة
- أزرار وصول سريع لكل قناة
- يُعرض **أعلى** لوحة التحكم لكافة المستخدمين

### 3. تحديث ترتيب الودجات في الصفحة الرئيسية

تعديل `DashboardLayout.tsx` أو الصفحة الرئيسية لعرض `CommunicationHubWidget` كأول عنصر.

### 4. تحديث عدادات الشارات

تعديل `useNotificationCounts.ts` لدعم badge موحد على مجموعة "التواصل والمشاركة" يجمع: رسائل + إشعارات + ملاحظات جديدة.

## الملفات المتأثرة

| ملف | نوع التغيير |
|-----|------------|
| `src/config/sidebarConfig.ts` | نقل المجموعة للترتيب 1 + إعادة تنظيم العناصر |
| `src/components/dashboard/widgets/CommunicationHubWidget.tsx` | **جديد** |
| `src/hooks/useNotificationCounts.ts` | تحديث التجميع |
| صفحة الداشبورد الرئيسية | إضافة الودجت في المقدمة |
