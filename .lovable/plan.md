## ملخص ما تم إنجازه

### قاعدة البيانات
- `consulting_offices` — بيانات المكتب القانونية والاعتمادات
- `office_consultant_memberships` — أدوار وصلاحيات الاستشاريين في المكتب
- `consultant_client_assignments` — ربط الاستشاري/المكتب بالجهات
- `office_signing_policies` — سياسات التوقيع لكل نوع مستند
- تطوير `consultant_document_signatures` — سياق المكتب والتضامن
- RLS شامل مع إصلاح الثغرات الأمنية

### لوحة الاستشاري الفرد (18 تبويب)
- نظرة عامة + إحصائيات + أزرار إجراء سريعة
- العمليات الميدانية + الاعتماد الفني + مركز المستندات
- الاعتمادات + التقارير الفنية + تقارير ESG
- التواصل القانوني + امتثال العملاء + مراجعة الشحنات
- التنبيهات + الجهات المرتبطة + عضويات المكاتب
- **الاعتماد والتوقيع** + **التراخيص** + **التفويضات** + **سجل الأنشطة**
- جهاتي + النقاط الخضراء + المساعد الذكي + مؤشرات KPI
- مفتاح تبديل النطاق (Organization Scope Selector)

### لوحة المكتب الاستشاري (10 تبويبات)
- نظرة عامة + أداء الفريق + لوحة المهام
- فريق العمل + العملاء + سياسات التوقيع + طلبات الاعتماد
- الامتثال + **المستندات** + **التراخيص** + **المالية** + **الإعدادات**
- المساعد الذكي

### منطق التوقيع المشترك
- `useConsultantCoSigning` — قاعدة التضامن الافتراضية
- سياسات توقيع قابلة للتخصيص من المدير
- موافقة المدير + ختم المكتب + نص التضامن

### الشروط والأحكام
- بنود مخصصة للاستشاري الفرد والمكتب
- المسؤولية التضامنية + إلزامية قبول الشروط

### الأمان

- تشديد سياسات `scoped_access_links` و`scoped_link_sessions`
