

# خطة: شريط تنقل سريع لأقسام لوحة التحكم

## الفكرة
إنشاء شريط تنقل ثابت (sticky) أعلى لوحة التحكم يعرض جميع الأقسام (المكونات الثابتة + التبويبات العشرة). عند الضغط على أي عنصر، يتم التمرير مباشرة إلى القسم المطلوب.

## التنفيذ

### ملف جديد: `src/components/dashboard/transporter/TransporterSectionNav.tsx`
- شريط أفقي قابل للتمرير (scrollable) مع تصميم مشابه لـ V2TabsNav
- يحتوي على 8 أقسام ثابتة + 10 تبويبات = 18 عنصر
- كل عنصر له أيقونة واسم مختصر
- عند الضغط يستخدم `scrollIntoView({ behavior: 'smooth' })` للانتقال للقسم
- يكون `sticky top-0` ليظل مرئياً أثناء التمرير
- يتتبع القسم المرئي حالياً عبر `IntersectionObserver` لتمييزه بصرياً

### تعديل: `src/components/dashboard/TransporterDashboard.tsx`
- إضافة `id` لكل قسم (مثل `id="section-header"`, `id="section-command"`, `id="section-tabs"` إلخ)
- إدراج `TransporterSectionNav` في أعلى الصفحة كـ sticky element
- عند الضغط على تبويب من الشريط، يتم التمرير + تغيير التبويب النشط في Tabs

### الأقسام في الشريط:
```text
┌──────────────────────────────────────────────────────────────┐
│ 🏠 الهيدر │ 🎯 القيادة │ ⚡ الإجراءات │ 📊 النبض │ 🔔 التنبيهات │
│ 💬 التواصل │ 📄 التوثيق │ ── التبويبات (10) ──────────────  │
└──────────────────────────────────────────────────────────────┘
```

### التقنيات:
- `IntersectionObserver` لتتبع القسم المرئي
- `element.scrollIntoView()` للتمرير السلس
- `sticky` positioning للثبات أعلى الصفحة
- نفس تصميم V2TabsNav (gradient, rounded, shadow)
