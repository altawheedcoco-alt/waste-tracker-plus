

# خطة: مكتبة أغاني حقيقية عبر Jamendo API (مجاني 100%)

## الحل

**Jamendo** — مكتبة موسيقية مجانية فيها **600,000+ أغنية حقيقية** من فنانين حقيقيين حول العالم، مرخصة Creative Commons. الـ API مجاني تماماً (35,000 طلب/شهر) ومفتاح الـ client_id عام (مش سري).

- أغاني عربية وشرقية ومصرية
- بوب، روك، هيب هوب، إلكتروني، جاز، كلاسيك
- كل أغنية فيها رابط استماع مباشر (MP3 stream)
- بحث بالاسم، الفنان، النوع، المزاج

## خطوات التنفيذ

### 1. Edge Function للبحث في Jamendo
- `supabase/functions/jamendo-search/index.ts`
- تستقبل: كلمة بحث، نوع (genre)، مزاج (mood)
- تستدعي `https://api.jamendo.com/v3.0/tracks/` مع `client_id`
- ترجع قائمة أغاني مع روابط الاستماع والمعاينة

### 2. تحديث MusicPicker بالكامل
- تاب جديد **"أغاني حقيقية 🎤"** بجانب المكتبة المدمجة الحالية
- حقل بحث نصي (بحث بالاسم أو الفنان)
- فلاتر بالنوع: بوب، شعبي، إلكتروني، هيب هوب، جاز، روك، عربي
- معاينة صوتية مباشرة (30 ثانية) من رابط Jamendo
- عرض اسم الأغنية + اسم الفنان + غلاف الألبوم

### 3. حفظ الأغاني المفضلة
- جدول `favorite_tracks` لحفظ الأغاني المفضلة للمستخدم
- قسم "المفضلة" في MusicPicker للوصول السريع

### 4. الاحتفاظ بالمكتبة المدمجة
- المكتبة الحالية (Web Audio) تبقى كخيار "موسيقى مدمجة" للي مش عايز يبحث

## التفاصيل التقنية

- **Jamendo client_id** مفتاح عام (مش سري) — يُخزن في الكود مباشرة
- التسجيل مجاني على developer.jamendo.com
- رابط المعاينة: `track.audio` (MP3 stream مباشر)
- رابط قصير: `track.shorturl`
- غلاف الألبوم: `track.album_image`
