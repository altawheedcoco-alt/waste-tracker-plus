# تحليل: أين تظهر "المنشورات" في الجهة الناقلة؟

المنشورات تظهر في **5 أماكن مختلفة** داخل المنصة، وكلها متاحة للجهة الناقلة. كل مكان له طبيعة ووظيفة مختلفة تماماً:

---

## 1. منشورات الجهة (Organization Posts) — الحائط الاجتماعي

**المسار:** `/dashboard/organization-profile?tab=posts`
**في السايدبار:** الهوية والمؤسسة → المنشورات


| الخاصية           | التفاصيل                                                                                                          |
| ----------------- | ----------------------------------------------------------------------------------------------------------------- |
| **الجدول**        | `organization_posts`                                                                                              |
| **من ينشر**       | الجهة نفسها                                                                                                       |
| **من يرى**        | أي شخص يزور صفحة الجهة                                                                                            |
| **أنواع المحتوى** | نص، صور، فيديو، مستندات، وسائط متعددة وغيرها وايضا الفويس نوت                                                     |
| **التفاعلات**     | إعجاب (likes)، تعليقات، مشاركة داخليا وخارجيا عامه وخاصه بالنظم المختلفه الموضوعه داخل المشروع فيما يخص المشاركات |
| **أزرار متاحة**   | إنشاء منشور، حذف، تثبيت/إلغاء تثبيت، تحميل وسائط متعددة                                                           |
| **ميزة فريدة**    | يظهر كـ "حائط فيسبوك" على الصفحة التجارية العامة للجهة (`/org/:slug`)                                             |


---

## 2. الصفحة التجارية (Business Page Preview)

**المسار:** `/dashboard/organization-profile?tab=business` أو `/org/:slug`


| الخاصية          | التفاصيل                                                         |
| ---------------- | ---------------------------------------------------------------- |
| **الجدول**       | `organization_posts` (نفس البيانات)                              |
| **الفرق**        | عرض المنشورات بتصميم "صفحة تجارية" شبيه بفيسبوك                  |
| **أزرار إضافية** | تبويبات (منشورات، حول، خدمات، بورتفوليو)، إعدادات مشاركة         |
| **ميزة فريدة**   | يعرض المنشورات + معلومات الجهة + الخدمات في واجهة موحدة احترافية |


---

## 3. تايم لاين الجهات المرتبطة (Partners Timeline)

**المسار:** `/dashboard/partners-timeline`
**في السايدبار:** الهوية والمؤسسة → تايم لاين الجهات المرتبطة


| الخاصية        | التفاصيل                                                    |
| -------------- | ----------------------------------------------------------- |
| **الجدول**     | `organization_posts` (لكن من جهات أخرى)                     |
| **من ينشر**    | الجهات المرتبطة بالناقل (مولدين، مدورين، إلخ)               |
| **من يرى**     | الناقل يرى منشورات شركائه فقط                               |
| **التفاعلات**  | إعجاب، تعليق، مشاهدة الملف التجاري للجهة                    |
| **ميزة فريدة** | تجميع أخبار الشركاء في مكان واحد — غير موجود في أي مكان آخر |


---

## 4. منشورات قنوات البث (Broadcast Posts)

**المسار:** `/dashboard/broadcast-channels`
**في السايدبار:** التواصل → البث


| الخاصية           | التفاصيل                                                                                           |
| ----------------- | -------------------------------------------------------------------------------------------------- |
| **الجدول**        | `broadcast_posts` (جدول مختلف تماماً)                                                              |
| **من ينشر**       | مالك القناة فقط                                                                                    |
| **من يرى**        | المشتركون في القناة                                                                                |
| **أنواع المحتوى** | نص، وسائط متعددة و صور وفيديو ومستندات و وغير ذلك وايضا الفويس نوت (حتى 100+ ملف)، روابط مع معاينة |
| **التفاعلات**     | ردود فعل (reactions متعددة)، تعليقات، إعادة توجيه (forward)، تثبيت                                 |
| **أزرار متاحة**   | نشر، تثبيت، حذف، تعديل، reactions بأنواع مختلفة                                                    |
| **ميزة فريدة**    | بث أحادي الاتجاه، 3 مستويات وصول، عداد مشاهدات، إحصائيات                                           |


---

## 5. منشورات المنصة (Platform Posts) — للقراءة فقط

**المسار:** `/posts` (صفحة عامة) و `/posts/:id` (تفاصيل المنشور)
**في الهيدر:** المحتوى → منشورات المنصة


| الخاصية        | التفاصيل                                                          |
| -------------- | ----------------------------------------------------------------- |
| **الجدول**     | `platform_posts` (جدول مختلف)                                     |
| **من ينشر**    | مدير النظام فقط (عبر `/dashboard/posts-manager`)                  |
| **من يرى**     | الجميع (عام)                                                      |
| **التفاعلات**  | إعجاب، عداد مشاهدات                                               |
| **ميزة فريدة** | نشر مجدول (1,653 منشور)، توليد صور AI تلقائي، 9 تصنيفات، هاشتاجات |
| **لا يمكن**    | الناقل لا يستطيع النشر — قراءة فقط                                |

## ملخص الفروقات الجوهرية

1. **منشورات الجهة + الصفحة التجارية** = نفس البيانات بعرض مختلف (حائط اجتماعي vs صفحة احترافية)
2. **تايم لاين الشركاء** = قراءة منشورات الآخرين (لا يوجد في أي مكان آخر)
3. **قنوات البث** = نظام مستقل تماماً بجدول مختلف، أقوى في التفاعلات (reactions + forward + views)
4. **منشورات المنصة** = محتوى رسمي من الإدارة، قراءة فقط للمستخدمين

ممكن تنفذ التعديل وتجمعلي الخمسه بتبويبات جديده في لوحه التحكم الجهه الناقله بحيث يكونوا ظاهرين لوحدهم في مكان مميز في اعلي لوحه التحكم
